### Docker

모든 버전은 DockerHub에서 가져올 수 있습니다:

```sh
docker pull carlasim/carla:X.X.X
```

가장 최근 릴리스는 "latest" 태그를 사용하세요:

```sh
docker pull carlasim/carla:latest
```
