# 커스텀 맵

CARLA에서 맵은 도시의 3D 모델과 도로망 정의를 포함합니다. 도로망은 [__OpenDRIVE__](https://www.asam.net/standards/detail/opendrive/) 표준을 통해 정의됩니다. CARLA는 다양한 용도로 바로 사용할 수 있는 다양한 맵을 기본적으로 제공합니다. 사용자는 또한 자신만의 맵을 만들어 CARLA에 로드할 수 있습니다. 다음 일련의 튜토리얼에서는 CARLA에 커스텀 맵을 만들고 로드하는 데 필요한 단계를 자세히 설명합니다.
